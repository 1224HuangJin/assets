## 🚀 使用方法

1. 将你的 GIF 文件直接拖入当前 `assets/images/sticker/` 文件夹。
2. 确保 `index.html` 文件已包含。
3. 访问上方网址，查看所有 贴图，点击“复制”按钮获取直链。
4. 在 Discord 或任何支持嵌入的聊天工具中粘贴链接，动图会自动播放。
